# Python Voice Calculator

## Project Overview
This is a simple Voice Calculator developed in Python. It performs basic arithmetic operations and speaks the calculated result using a text-to-speech (TTS) library.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Voice output of results
- Easy to use

## Technologies Used
- Python
- pyttsx3 (Text-to-Speech)
- VS Code

## How to Run
1. Install Python.
2. Install pyttsx3:
   pip install pyttsx3
3. Run the Python file:
   python calculator.py
